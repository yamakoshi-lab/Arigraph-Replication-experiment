# AriGraph 評価スクリプトの実行方法

本番環境の評価、および検証用スクリプト（200問テスト等）をローカルで実行する際の環境構築・実行手順です。

## 1. 必須環境の読み込み

実行する前に、必ず仮想環境の有効化と、APIキーなどの環境変数を読み込む必要があります。
特に `GEMINI_API_KEY`（OpenAI APIへアクセスするためのキー）は `~/.bashrc` に保存されているため、以下のコマンドで必ずソース化（反映）してください。

```bash
# 1. bashrcからAPIキーを読み込む
source ~/.bashrc

# 2. Pythonの仮想環境を有効化する
source /home/user/研究2026-06-06/.venv/bin/activate

# 3. モジュールインポート用のPYTHONPATHをルートディレクトリに設定する
export PYTHONPATH=.
```

## 2. 実行コマンド

上記の環境設定が完了した状態で、以下のスクリプトを実行します。

### 検証用高速テストプログラムの実行
重複排除済みグラフを使用し、閾値を指定して200問を一気に検証するスクリプトです。
（※現在、決定論的スコアを取得するために内部のQA生成温度は `Temperature=0.0` に設定されています）

```bash
# 引数に閾値（例: 0.65）を渡して実行します
python3 scratch/test_threshold_fast.py 0.65
```

### 実行時の注意点
- OpenAI APIへ大量の通信を行うため、完了までに通常30分〜40分ほどかかります。
- バックグラウンド実行したい場合は、`nohup python3 scratch/test_threshold_fast.py 0.65 > output.log 2>&1 &` などを使用してください。
- 途中でAPIキーが無効になったり、Rate Limitに引っかかった場合は、回答が「unknown」として処理され、F1スコアが0%に落ち込むことがあります。
